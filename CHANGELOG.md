# Changelog

All notable changes to the SpectrumX UI Library will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- Initial release of SpectrumX UI Library
- Complete set of UI components:
  - Toggle with smooth animations
  - Button with multiple styles (default, accent, warning, info)
  - Text Input with placeholder support
  - Number Input with min/max constraints
  - Slider with draggable knob
  - Dropdown (single select)
  - Multi Dropdown with priority ordering
  - Checkbox
  - Label
  - Section titles
  - Status Card with loading animation
  - Separator
- Draggable main window
- Draggable status cards
- Floating toggle button
- Tab system with letter/emoji icons
- Notification system with 4 types (info, success, warning, error)
- Mobile-optimized interface
- Smooth tween animations
- Theme system with 8 pre-made themes:
  - Red (default)
  - Blue
  - Purple
  - Green
  - Orange
  - Pink
  - Dark (monochrome)
  - Midnight (deep blue)
- Comprehensive documentation

### Features
- Touch-friendly for mobile devices
- Left/Right column layout in tabs
- Proper z-index handling for dropdowns
- Auto-resizing scroll frames
- Pulsing animations for status indicators
- Programmatic control of all components
- Easy-to-use API
